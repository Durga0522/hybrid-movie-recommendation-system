# Hybrid Movie Recommendation System

This project is a Hybrid Movie Recommendation System built using Python, Streamlit, Scikit-learn, and Surprise.

## Features
- Content-Based Filtering
- Collaborative Filtering
- Hybrid Recommendation
- Popular Movies Section
- Interactive Streamlit UI

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Surprise
- Streamlit

## Files
- app.py
- recommender.py
- movies.csv
- ratings.csv
- requirements.txt

## Run Locally

1. Install dependencies:
   pip install -r requirements.txt

2. Start the app:
   streamlit run app.py
